1. 연관분석 개요
데이터들 사이의 자주 발생하는 속성을 찾고, 그 속성들 사이의 연관성이 어느 정도 있는 분석
활용분야 : 상품진열, 사기보험적발, 신상품 카테고리 구성 ...

조건(left-hand-side, item_base) : input_data(x) => 결과(right-hand-side, item_add) : output_data(y)


연관분석의 지표
1. 지지도(support) : 전체 데이터 중에서 조건, 결과 항목들이 포함된 거래 비율(얼마나 자주 함께 나타나는지)
    (x, y)의 항목 수 / 전체데이터 수 = 0.2
    
2. 신뢰도(confidence) : 조건이 발생 했을 때, 결과가 동시에 일어날 확률(조건이 오면 결과가 얼마나 자주 나오는지)
    (x->y)의 항목 수 / x가 나오는 항목 수 = 0.5

3. 향상도(lift) : 우연히 발생할 규칙은 아니였는지 확인
    <1 : 함께 나타날 가능성이 기대보다 낮다
    >1 : 양의 상관관계
    
    x->y의 지지도 / (x의 지지도*y의 지지도) = 0.2 / (0.4*0.6) = 0.833
