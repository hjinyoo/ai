font 불러오기
@font-face {
    font-family: '폰트이름';
    src: url(폰트위치주소);}
전체 폰트 지정
* {
    font-family: '폰트이름','폰트이름';
}

display 속성
화면에 어떻게 보이는지를 설정하는 속성. 
block : div태그 p태그 li태그 - 높이의 속성o
inline : 옆으로 붙여 나가는 것 - 높이의 속성x
inline-block : block의 속성을 갖고 옆으로 붙여 나가는 것 - 높이의 속성
none : 안 보이게 함

visibility 속성
display 속성의 none속성값과 비교하여 이해
-visibility:hidden
공간은 확보, 안보임,기본은불투명
-display:none
공간도 사라짐
-opacity 속성 : 투명도를 조절하는 속성

margin 및 padding 속성
정확한 속성의 의미를 알아야 한다
margin : 밖여백
padding : 안여백

box-sizing 속성
box-sizing속성을 이용하면 전체 사이즈의 크기를 변경
box-sizing: border-box; /*border를 width안으로*/

글자 관련된 속성들
font-family(글꼴), font-size(크기), font-style(이텔릭체), font-weight(진하게), line-height(행간격 조정용도나 세로축에 대한 정렬 용도), text-align(글자에 대한 정렬), text-decoration(간단한 글꼴 모양의 속성) 
line-height: height의 사이즈 ; /*수직 가운데(height와 line-height를 같게*/


폰트(웹폰트,로컬폰트, 시스템폰트)
웹 폰트 : 웹 폰트를 다운받아 웹사이트를 구성 (ex) http://fonts.google.com
로컬폰트 : 구글이나 네이버 폰트, 그외 사이트

position 속성
-absolute 속성값 
position: absolute; /*절대좌표*/
-fixed 속성값
position: fixed; /*지정된 위치에 고정(스크롤 등 움직여도 화면상 고정*/
-relative 속성값 
position: relative; /*상대좌표(원래 있을 위치 기준)*/
z-index: ; /*z-index가 높은 값이 위로*/

float 속성
position 속성과 함께 요소의 위치를 설정하기 위한 속성
ex)
#container1 img{float: left;} /*이미지 왼쪽으로*/
#container2 img{float: right;} /*이미지 오른쪽으로*/
