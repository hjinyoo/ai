선택자
HTML 문서 중 특정 부분을 선택하여, 해당 부분의 속성을 변경하는 목적으로 사용 (CSS를 사용하는 이유, Javascript, jQuery등에서도 사용
-tag 선택자
특정 태그를 선택하여 CSS속성을 적용

-* 선택자
전체라는 의미의 * 선택자를 사용하면 문서 전체를 선택하는 의미
-id(#)와 class(.) 
태그의 id와 class를 이용하여 CSS에 속성을 설정
tag, id, class를 혼합하여 CSS에서 속성을 설정할 수 있다.

-속성 선택자 
tag의 속성을 선택하여 CSS의 속성을 설정
선택자[속성] : 지정한 속성을 가지고 있는 요소를 찾아 스타일을 적용
선택자[속성=값] : 속성 값이 특정값과 같은 경우를 찾아 스타일을 적용
선택자[속성^=값] : 속성 값이 지정한 문자로 시작하는 속성값에 대해서만 스타일을 적용
선택자[속성$=값] : 속성 값이 지정한 문자로 끝나는 속성에 대해서만 스타일을 적용

-후손 및 자손 선택자
tag의 속성을 선택하여 CSS의 속성을 설정 

-동위(=형제) 선택자(+하나만, ~다선택)
태그의 동등한 위치를 판단하여 CSS속성을 설정
h3 ~ div { /*h3태그의 동위 선택자들 다(~)*/}
h3 + div{ /* h3태그의 동위선택자들 중 바로 다음 것*/}

-반응 선택자
마우스를 올려 놓았을 때(hover상태)의 반응에 따른 속성을 설정
특정태그:hover{ /* li위에 마우스를 올렸을 때 반응할 디자인 (반응선택자)*/

-상태 선택자
상태에 따라 CSS속성이 변화하는 설정 
ex. input태그의 경우 focus가 맞혀졌을 때
        input:focus{ /*상태선택자*/}

input:disabled{ } : disabled 속성이 적용되어 비활성화된 <input>요소 선택하여 상호작용을 할수없는 입력 필드의 스타일 지정
input:enabled{ } : disabled 속성이 없어 활성화된 <input>요소를 선택하여 값을 입력하거나 변경할 수 있는 입력 필드의 스타일을 지정
input:read-only{ }: read-only속성이 적용되어 읽기 전용인 <input> 요소를 선택하여 볼수는 있지만 직접 수정할 수 없는 입력 필드의 스타일을 지정

-구조 선택자
구조에 따라 CSS 속성이 변화하는 설정
ex) 
#nav li:nth-child(2n+1){ /*#nav 밑의 홀수번째 li(첫번째)*/ }
#nav li:first-child{ /*#nav밑의 첫번째 li*/}

세로를 가로로 나오게 하기
#nav ul {overflow: hidden;} /* li를 가로로할 단의 부모단*/
#nav ul li {float: left; }  /* 가로로할 단/

border-radius: 0 0 0 0; /*좌상, 우상, 우하, 좌하*/ 둥글게 만들기

-링크 선택자
문서에 링크(href) 되어 있는 문자를 선택하여, CSS 속성을 설정
ex)
#링크가있는태그명 a:after{ /* 링크선택자 */}

-부정 선택자
나를 제외한 모든 태그에 CSS 속성을 적용
ex)
#li가있는태그명 li:not(.web) a{ /*부정선택자:li중에 .web을 제외하고*/}

display:block; /* 블록형 태그로 */
