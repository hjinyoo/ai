Sequence
시퀀스의 정의 
순차적인 번호를 자동으로 생성하는 객체로 테이블과 독립적으로 생성 및 저장 가능
특징
퀀스에서 생성되는 번호는 유일하기 때문에 기본 테이블에서 인조 Primary Key 생성시 주로 사용
여러 테이블에 의해 공유도 가능
시퀀스는 테이블과 관계없이 생성, 저장. 오라클 내부 루틴에 의해 발생되고 증가, 감소
문법 : CREATE SEQUENCE sequence_name 
[INCREMENT BY n]  - 기본값 1
[START WITH n]      - 기본값 1
[MAXVALUE n]       - 최대값
[MINVALUE n]        - 최소값
[CYCLE | NOCYCLE] - 반복 여부
[CACHE n | NOCACHE]; - 캐시 사용 여부

DUAL : 오라클 제공 테이블(1행 1열)
SELECT sequence_name .NEXTVAL FROM DUAL; -- NEXTVAL(순차번호 자동생성)
SELECT sequence_name .CURRVAL FROM DUAL; -- CURRVAL(현재까지 진행된 순차번호)

시퀀스 삭제
DROP SEQUENCE sequence_name;

시퀀스 수정
ALTER SEQUENCE sequence_name [INCREAMENT BY n] [START WITH n] [MAXVALUE n][MINVALUE n];
