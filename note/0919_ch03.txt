3장 데이터 구조: 여러 데이터를 한 변수에 담는 목적
1절.리스트:[타입과 관련 없는 0개 이상의 데이터들]
	1.1리스트 생성 : [], list(), range(from, to, step)
	1.2기본 정보 조회
		-min(lst), max(lst), sum(lst), len(lst)
		-sorted(lst)
		-lst.sort() : 원본이 정렬데이터로 변경
		-lst.reverse()
	1.3요소 추가 : append(value), extend([values]), insert(idx, value)
	1.4인덱싱과 요소 개수
		lst[idx]
		lst.index(찾을 데이터) : 없는 데이터를 찾을 경우 Error
		찾을 데이터 in list
		lst.count(찾을 데이터)
