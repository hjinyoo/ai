이벤트
이벤트 관련 용어 정리
이벤트란 키보드를 이용해 버튼을 입력하거나 마우스 클릭과 같이 다른 것에 영향을 미치는 것. 

사전적 정의 : 자바 스크립트에서 DOM을 통해 동적인 화면처리를 위해 처리하는 내용
마우스 이벤트, 키보드 이벤트, HTML 프레임 이벤트, HTML 입력 양식 이벤트, 유저 인터페이스 이벤트, 구조 변화 이벤트, 터치 이벤트 등이 있다
(ex) onclick, onmouseover, onmouseout, onkeyup(키보드의 특정키를 눌렀다가 올라올 때, event.keyCode 해당키의 고유 코드값..)
­	이벤트를 연결한다. 
-Window 객체의 onload 속성에 함수 자료형을 할당하는 것
-load는 이벤트 이름, 이벤트 타입
-header 객체의 이벤트 이름은 click, 이벤트 속성은 onclick, 이벤트리스너는 whenClick()함수
-문서 객체에 이벤트를 연결하는 방법을 이벤트 모델

인라인 이벤트 모델
HTML 요소안에 속성을 사용하여 자바스크립트 코드를 직접 삽입(onclick, onmouseover, onload 등)

기본 이벤트 모델
HTML 요소 객체의 이벤트 핸들러 프로퍼티에 직접 함수를 할당하여 이벤트 처리
기본이벤트모델(고전적 이벤트 모델) : 이벤트 속성에 function 할당
on으로 시작하는 속성 : 이벤트 속성
이벤트 속성변수에 할당된 함수 : 이벤트 리스너 함수(this 사용시 화살표 함수 불가)

표준 이벤트 모델
현재 웹개발에서 일반적으로 사용되는 이벤트 처리 방식
addEventListener() 메소드와 removeEventListener()메소드를 사용하여 이벤트 등록,제거하는 방식 기반

this
this는 자바스크립트에서 호출된 방식에 따라 그 값이 결정되는 키워드
※ 화살표 함수 사용시 사용불가

이벤트 강제 실행
직접적인 조작없이 자바스크립트 코드를 통해 특정 HTML요소안에 이벤트를 인위적으로 발생시키는 것

이벤트 제거
HTML 요소에 등록된 이벤트 리스너를 더이상 실행되지 않도록 해제하는 과정
removeEventListener('',handler); 이용
