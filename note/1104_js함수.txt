매개변수와 리턴값
function 함수이름(매개변수, 매개변수, 매개변수) {
	//함수 코드
	return 리턴값;
}
선언된 매개변수보다 많이 매개변수로 호출될 경우 : 뒷부분 무시
선언된 매개변수보다 적은 매개변수로 호출될 경우 : 전달되지 않은 매개변수는 undefined

가변 인자 함수
매개변수를 선언된 형태와 다르게 사용했을 때, 매개변수를 모두 활용할 수 있는 함수. 매개변수의 개수가 변할 수 있는 함수(파이썬 튜플매개변수로 구현) ex. Array()함수
※ Array()함수 ; 빈 배열 생성
Array(number) ; 매개변수만큼의 크기를 가지는 배열을 만든다.
Array(any, …, any); 매개변수를 배열로 만든다.
arguments : 매개변수 배열(매개변수의 내용)
 * 매개변수가 0개 : length가 0인 배열 return
 * 매개변수가 1개 : length가 매개변수인만큼의 크기인 배열 return
 * 매개변수가 2개  이상 : 매개변수로 배열을 생성

내부 함수
함수 내부에 선언하는 함수
충돌을 막기 위해 내부함수 사용하면 내부함수가 우선
 jQuery는 대부분 선언적 함수를 내부함수로 작성하고 있다

콜백함수
매개변수에 넣어 전달된 함수 
자바스크립트에서는 함수도 하나의 자료형이므로 매개변수에 넣을 수 있다.
※ 콜백함수는 익명함수로 사용하는 경우도 많다.

타이머 함수
setTimeout(function, millisecond) : 일정시간 후 함수를 한번 실행
setInterval(function, millisecond) : 일정시간마다 함수를 반복해서 실행
clearTimeout(id) : setTimeout 중지
clearInterval(id) : setInterval 중지
※자바스크립트 함수 중에 브라우저에 처리를 부탁하는 함수인 타이머 함수는 웹 브라우저가 처리하고 처리가 완료되었다는 것을 자바 스크립트에게 알려주는 방식이다. 이러한 함수는 현재 단위가 끝나기전에는 실행되지 않는다.
